# Event Booking Platform

# Stack used 
- React + Tailwind 
- React Router
- TanStack Query
- Gin (GoLang)
- Redis (Memory DB)
- NATS (Message Queue)
- S3 & CloudFront (AWS)
- MySql (Disk DB)
- GoPDF (pdf generation)
